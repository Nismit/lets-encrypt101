Let's Encryptとは

無料で使えるSSL(Secure Sockets Layer) 証明書

SSLって何

HTTP/HTTPS

  • Let's Encryptが出る前まで基本有料
  • シマンテック、グローバルサインなどが有名
  • 料金はピンキリ

SSL証明レベル

  • レベル1: DV(Domain Validation) SSL
  • レベル2: OV(Organization Validation)SSL
  • レベル3: EV(Extended Validation)SSL

Let's Encryptの目的

より安全なインターネット通信の普及

Let's Encrypt 基本方針

  • 無料
  • 運用自動化
  • 安全性
  • 透明性

Internet Security Research Group(ISRG)
アメリカの公益法人

無料の理由

スポンサーによる支援や寄付によって運用

  • mozilla
  • akamai
  • cisco
  • facebook etc..

認証の自動化などもコスト削減に繋がっている

デメリット

  • レンタルサーバの対応率が低い
  • 3ヶ月毎の更新が必要
  • ドメイン認証SSLレベルである
  • トラブルの対処

メリット

  • HTTP2が使える
  • GoogleがSSL対応サイトの優遇
  • サイトの信頼度が増加
